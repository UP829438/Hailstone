HTML code

Side note: All elements of HTML code must have a closing tag too. So no matter what tag, you will need to have the </> of the same tag when it needs to be closed.

<!DOCTYPE html>

This is how all HTML code documents should start. This lets the page know that all content of the page is written in HTML code and needs to be understood as such.

<head>

The head tag is the tag which contains all the important declarations for the website rather than actual content. Elements normally included in the head tag include linking to style sheets, declaring the text format, the title of the page and more.

Example of head tag in use:


<head>
    <link rel="icon" type="image/png" sizes="16x16" href="img/favicon.ico">
    <meta charset="UTF-8">
    <link rel="stylesheet" type="text/css" href="main.css">
    <link rel="stylesheet" type="text/css" href="style3.css">
    <link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet">
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js" type="text/javascript"></script>
    <script type="text/javascript" src="scripts.js"></script>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Discovery Log</title>
</head>

<link>

The link tag is what allows the page to be linked to other pages. This is most commonly done to link a page to a CSS style sheet, separating form and content.

Example of link tag in use:


          <link rel="stylesheet" type="text/css" href="main.css">

<script>

I have used this tag within my website in order to link to a javascript file, which stores different functions that help make the website more aesthetically pleasing.

Example of script tag:


          <script type="text/javascript" src="scripts.js"></script>

<meta>

The meta tag provides metadata about your HTML page. This can include different things such as setting the text format for the page.

Example of meta tag:


          <meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>

The title tag allows you to set the title of the page you are on. If you look closely, you will find that the title of this page is Discovery Log, shown on the selected tab at the top of the browser.

Example of title tag:


          <title>Discovery Log</title>

<body>

The body tag is the main part of your webpage. This will include all of the content that the web page reads and outputs it to the page without any formatting (unless you use a <style> element within the body which is bad practice).

Example of body tag: (greatly reduced for spacial reasons)


          <body>
            <div class="container">
              <p class="quicklink">Quick links:</p>
              <div class="sidedrop1">
                <p>Weeks <button onclick="rundrop1()" class="weekdrop1"><img src="img/arrow.svg" alt="#"></button></p>
          </body>

class

Declaring a class within the HTML code allows you to reference just the class directly when changing the appearance using CSS. This allows for the actual element to stay in standard form if any other content uses the same element.

<nav>

This tag is used to declare anything to do with navigation. Most websites have navigation bars which allow the user to navigate through the website. If you have a site with multiple pages, the chances are that you will need a navigation bar of some sort to allow ease of use. The nav bar should have a similar format throughout different pages.

Example of nav tag:


          <nav>
              <ul>
                  <li class="dropdown">
                      <button aria-label="Menu" onclick="runDropDown()" class="dropmenu"><img class="dropimg" src="img/drawing.svg" alt="" id="image"></button>
                      <div id="drop" class="dropdowncontent">
                          <a href="index.html">Home</a>
                          <a href="litreview.html">Literature review</a>
                          <a class="current" href="discovlog.html">Discovery Log</a>
                          <a href="video.html">Video</a>
                      </div>
                      <li class="title">
                          <h1>Discovery Log</h1> </li>
              </ul>
          </nav>

<ol>

The ol tag allows you to create an ordered list, in other words a numbered list.. In order to create list elements within this tag, use <li>(your text here)</li>

Example of ol tag:


          <ol>
            <li>item 1</li>
            <li>item 2</li>
            <li>item 3</li>
          </ol>

<ul>

The ul tag functions very similarly to the ol tag, however the list is unordered so by default, any li elements within this will be bullet points.

Example of ul tag:


          <ul>
            <li>item 1</li>
            <li>item 2</li>
            <li>item 3</li>
          </ul>

<div>

The div tag allows you to separate content into different boxes essentially. The div tag is very useful when creating the layout of your web page.

Example of div tag:


          <div class="reviewtext">
              <h2 class="reviewtitle">Preventing Data Leakage on the web</h2>
